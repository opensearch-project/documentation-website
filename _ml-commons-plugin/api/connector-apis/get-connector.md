# Get a connector

This API retrieves a connector by its ID.

### Path and HTTP methods

```json
GET /_plugins/_ml/connectors/<connector_id>
```

#### Example request

```json
GET /_plugins/_ml/connectors/N8AE1osB0jLkkocYjz7D
```

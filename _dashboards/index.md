# OpenSearch Dashboards

OpenSearch Dashboards is the web UI for [OpenSearch]({{site.url}}{{site.baseurl}}/getting-started/intro/). You can use OpenSearch Dashboards to perform most tasks you can do with the OpenSearch APIs. You can also create visualizations and data dashboards[^1] with OpenSearch Dashboards.

[^1]: The terminology is confusing. _OpenSearch Dashboards_ or _Dashboards_ (plural) refers to the OpenSearch UI. **Dashboard** (singular, bold) refers to the dashboard-building application in Opensearch Dashboards. The term _dashboard_ (lower case) refers to a web dashboard, a collection of data visualizations that you can create in **Dashboard**.

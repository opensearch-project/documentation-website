# Style options

The visualization editor provides shared style settings that appear across multiple chart types. These settings are displayed as collapsible panels in the chart settings sidebar.

- [Thresholds]({{site.url}}{{site.baseurl}}/dashboards/visualize/visualization-editor/style-options/thresholds-setting/) — Define color-coded boundaries to visually indicate when values cross important limits.
- [Value calculations]({{site.url}}{{site.baseurl}}/dashboards/visualize/visualization-editor/style-options/value-calculations/) — Choose how a series of values is reduced to a single number for display.
- [Value mappings]({{site.url}}{{site.baseurl}}/dashboards/visualize/visualization-editor/style-options/value-mappings/) — Replace raw values with custom labels and colors.

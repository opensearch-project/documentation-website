# Configuring maps

You can configure map tile servers used by [coordinate maps]({{site.url}}{{site.baseurl}}/dashboards/visualize/coordinate-maps/) and the [Maps application]({{site.url}}{{site.baseurl}}/dashboards/visualize/maps/). These settings do not apply to [region maps]({{site.url}}{{site.baseurl}}/dashboards/visualize/region-maps/), which use vector boundaries (GeoJSON files) rather than raster tiles.

## Related documentation

- [Configuring a Web Map Service (WMS)]({{site.url}}{{site.baseurl}}/dashboards/visualize/maptiles/) -- Configure a WMS tile server for custom basemaps.
- [Using self-hosted map servers]({{site.url}}{{site.baseurl}}/dashboards/visualize/selfhost-maps-server/) -- Host map tiles locally for air-gapped environments.

# API Token APIs
**Introduced 3.7**
{: .label .label-purple }

The API Token APIs let security admins create, list, and revoke API keys for programmatic access to OpenSearch.

All endpoints require security admin privileges.

## Endpoints

- [Create API Token]({{site.url}}{{site.baseurl}}/api-reference/security/api-tokens/create/)
- [List API Tokens]({{site.url}}{{site.baseurl}}/api-reference/security/api-tokens/list/)
- [Revoke API Token]({{site.url}}{{site.baseurl}}/api-reference/security/api-tokens/revoke/)

# API Key APIs
**Introduced 3.7**
{: .label .label-purple }

The API Key APIs let security admins create, list, and revoke API keys for programmatic access to OpenSearch.

All endpoints require security admin privileges.

## Endpoints

- [Create API Key]({{site.url}}{{site.baseurl}}/api-reference/security/api-keys/create/)
- [List API Keys]({{site.url}}{{site.baseurl}}/api-reference/security/api-keys/list/)
- [Revoke API Key]({{site.url}}{{site.baseurl}}/api-reference/security/api-keys/revoke/)
